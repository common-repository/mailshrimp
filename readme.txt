== Description ==

MailShrimp is a simple plugin for using your MailChimp account, right from your blog.

With MailShrimp, you'll be able to send easily your campaigns, just as if you were doing it from MailChimp.

== Installation ==

1. Upload the `mailshrimp` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set your MailChimp API key from the Settings page
